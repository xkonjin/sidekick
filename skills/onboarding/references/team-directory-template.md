## How to Add People

Tell Sidekick "add [name] to my team directory" or edit this file directly.

People get added automatically when:

- They appear in a meeting transcript Sidekick processes
- You mention someone new and provide context
- A new project introduces new collaborators

## Relationship Types

| Type               | Meaning                               |
| ------------------ | ------------------------------------- |
| Reports to you     | Direct report                         |
| You report to them | Your manager or skip-level            |
| Peer               | Same level, collaborate regularly     |
| Stakeholder        | Needs to be informed or consulted     |
| External           | Vendor, partner, client               |
| Cross-functional   | Different team, regular collaboration |
