# People Management Skill Pack

Skills for managers, team leads, and anyone who works closely with people. Covers 1:1s, hiring, team health, feedback, and capacity planning.

## Skills (6)

| Skill                   | What It Does                                                  |
| ----------------------- | ------------------------------------------------------------- |
| 1:1 Prep                | Prepares agendas for one-on-one meetings using recent context |
| Team Health Check       | Structured team health assessment with anonymized patterns    |
| Hiring Brief            | Creates role descriptions from team needs and gap analysis    |
| Performance Review Prep | Gathers evidence and structures performance conversations     |
| Feedback Session        | Structures giving and receiving feedback using SBI framework  |
| Team Capacity Planner   | Maps team capacity against upcoming work                      |

## Install

Append the contents of `INDEX.md` to the Skills Index table in your CLAUDE.md.
