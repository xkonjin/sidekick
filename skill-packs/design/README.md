# Design Skill Pack

4 skills for design review and documentation: heuristic evaluation, accessibility audits, component specs, and design system docs.

## Skills

| Skill               | What it does                                                                  |
| ------------------- | ----------------------------------------------------------------------------- |
| design-review       | Heuristic evaluation: consistency, hierarchy, accessibility, alignment        |
| design-system-docs  | Component inventory with props, variants, usage guidelines, do/don't examples |
| accessibility-audit | WCAG 2.1 AA compliance: contrast, labels, keyboard nav, screen reader, focus  |
| component-spec      | Technical spec bridging design to implementation: props, states, edge cases   |

## Install

Say "add the design skills" during any conversation, or manually:

1. Open your `CLAUDE.md`
2. Find the `## Skills` table
3. Append the contents of `INDEX.md` to the table
4. Skill files in `skill-packs/design/skills/` load on-demand
