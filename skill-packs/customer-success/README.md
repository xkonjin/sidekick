# Customer Success Skill Pack

5 skills for account health monitoring, QBRs, escalation handling, voice of customer synthesis, and success playbooks.

## Skills

| Skill              | What it does                                                            |
| ------------------ | ----------------------------------------------------------------------- |
| account-health     | Health card with usage signals, relationship warmth, risk flags.        |
| qbr-builder        | QBR deck outline: value delivered, milestones, expansion opportunities. |
| escalation-handler | Escalation brief with severity, comms plan, and resolution tracking.    |
| voice-of-customer  | Themed feedback synthesis from accounts and meeting notes.              |
| success-playbook   | Repeatable milestone and touchpoint plan by customer segment.           |

## Install

Say "add the customer success skills" during any conversation, or manually:

1. Open your `CLAUDE.md`
2. Find the `## Skills` table
3. Append the contents of `INDEX.md` to the table
4. Skill files in `skill-packs/customer-success/skills/` load on-demand
