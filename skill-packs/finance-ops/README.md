# Finance & Operations Skill Pack

5 skills for budget tracking, vendor evaluation, process documentation, cost analysis, and operational runbooks.

## Skills

| Skill            | What It Does                                                                    |
| ---------------- | ------------------------------------------------------------------------------- |
| budget-tracker   | Living budget: categories, allocated vs spent, burn rate, and projections       |
| vendor-evaluator | Requirements matrix, scoring, pricing comparison, and buy/build recommendation  |
| process-mapper   | Business process documentation: steps, owners, tools, handoffs, SLAs            |
| cost-analyzer    | Cost-benefit analysis: direct costs, hidden costs, payback period, alternatives |
| ops-runbook      | Operational runbook: triggers, steps, escalation paths, known pitfalls          |

## Install

Append the contents of `INDEX.md` to the Skills Index table in your CLAUDE.md.
