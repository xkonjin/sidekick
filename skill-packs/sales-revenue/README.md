# Sales & Revenue Skill Pack

6 skills for pipeline management, discovery prep, proposals, objection handling, win/loss analysis, and forecasting.

## Skills

| Skill             | What it does                                                            |
| ----------------- | ----------------------------------------------------------------------- |
| deal-tracker      | Living pipeline: deal cards with stage, contacts, next steps, blockers. |
| discovery-prep    | Research + call script for first meetings with prospects.               |
| proposal-builder  | Structured proposal outline with scope, timeline, and pricing.          |
| objection-bank    | Living library of objections with evidence-backed responses.            |
| win-loss-analysis | Post-deal review capturing what worked, what didn't, and patterns.      |
| forecast-check    | Weighted pipeline math with risk flags and gap-to-target.               |

## Install

Say "add the sales revenue skills" during any conversation, or manually:

1. Open your `CLAUDE.md`
2. Find the `## Skills` table
3. Append the contents of `INDEX.md` to the table
4. Skill files in `skill-packs/sales-revenue/skills/` load on-demand
