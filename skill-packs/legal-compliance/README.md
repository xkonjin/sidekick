# Legal & Compliance Skill Pack

4 skills for contract review, compliance checks, risk management, and policy drafting.

## Skills

| Skill                | What It Does                                                                  |
| -------------------- | ----------------------------------------------------------------------------- |
| contract-reviewer    | First-pass contract review: key terms, unusual clauses, missing protections   |
| compliance-checklist | Maps requirements for a given regulation or standard against current state    |
| risk-register        | Living risk register with probability/impact matrix and mitigation tracking   |
| policy-drafter       | Internal policies with purpose, scope, rules, exceptions, and review schedule |

## Install

Append the contents of `INDEX.md` to the Skills Index table in your CLAUDE.md.
