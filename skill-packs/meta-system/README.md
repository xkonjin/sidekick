# Meta & System Skill Pack

8 skills for managing how Sidekick works: workflow automation, skill discovery, template creation, context handoffs, daily planning, weekly reflection, assumption testing, and progress tracking.

## Skills

| Skill                  | What it does                                                                          |
| ---------------------- | ------------------------------------------------------------------------------------- |
| workflow-builder       | Creates reusable workflows by chaining existing skills with triggers and conditions.  |
| skill-suggester        | Maps any request to the best matching skill or chain, including uninstalled packs.    |
| template-factory       | Extracts any output you liked into a reusable template saved to your sidekick.        |
| context-handoff        | Generates a comprehensive context document for project or role handoffs.              |
| day-planner            | Reads calendar, tasks, and priorities to produce a focused daily plan.                |
| weekly-review          | Deeper than weekly-update: combines accomplishments with lessons, mistakes, patterns. |
| assumption-stress-test | Systematically attacks a plan to find fragile assumptions and blind spots.            |
| progress-radar         | Scans projects and priorities against timelines, produces a red/yellow/green radar.   |

## Install

Say "add the meta and system skills" during any conversation, or manually:

1. Open your `CLAUDE.md`
2. Find the `## Skills` table
3. Append the contents of `INDEX.md` to the table
4. Skill files in `skill-packs/meta-system/skills/` load on-demand
