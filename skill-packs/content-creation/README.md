# Content Creation Skill Pack

Skills for content marketers, writers, and anyone who creates written content regularly. Covers blog posts, social media, newsletters, content briefs, and repurposing.

## Skills (5)

| Skill                 | What It Does                                                  |
| --------------------- | ------------------------------------------------------------- |
| Blog Post Drafter     | Structures and drafts blog posts matched to your voice        |
| Social Media Calendar | Plans social content across platforms with scheduling         |
| Newsletter Builder    | Creates newsletter editions from recent activity and content  |
| Content Brief         | Writes briefs for content assignments (internal or freelance) |
| Repurpose Content     | Transforms existing content into new formats                  |

## Install

Append the contents of `INDEX.md` to the Skills Index table in your CLAUDE.md.
