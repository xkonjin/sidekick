# Automation Skill Pack

5 skills for operational automation: drafting, meeting prep, blocker audits, vendor tracking, and status reporting.

## Skills

| Skill          | What it does                                                                |
| -------------- | --------------------------------------------------------------------------- |
| email-drafter  | Context-aware email drafts matched to your actual tone. Never sends.        |
| meeting-prep   | Attendee context, agenda items, talking points, open questions to raise.    |
| blocker-audit  | Scans tasks for stalled items, flags escalation-worthy blockers.            |
| vendor-tracker | Multi-vendor status table: project, last contact, next milestone, blockers. |
| status-report  | Formatted progress update: done, in-progress, blocked, next.                |

## Install

Say "add the automation skills" during any conversation, or manually:

1. Open your `CLAUDE.md`
2. Find the `## Skills` table
3. Append the contents of `INDEX.md` to the table
4. Skill files in `skill-packs/automation/skills/` load on-demand
