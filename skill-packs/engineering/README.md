# Engineering Skill Pack

6 skills for engineering workflows: code review, architecture decisions, incident analysis, technical debt, documentation, and testing.

## Skills

| Skill             | What it does                                                                      |
| ----------------- | --------------------------------------------------------------------------------- |
| code-review       | Quality, security, performance, and readability analysis with line-level feedback |
| adr               | Architecture Decision Record: context, decision, consequences, alternatives       |
| postmortem        | Timeline, root cause, contributing factors, action items, lessons learned         |
| tech-debt-tracker | Categorize, score effort vs impact, prioritize technical debt items               |
| api-docs          | Endpoint docs: method, path, params, request/response examples, errors            |
| test-strategy     | Test pyramid, coverage priorities, edge cases, integration test plan              |

## Install

Say "add the engineering skills" during any conversation, or manually:

1. Open your `CLAUDE.md`
2. Find the `## Skills` table
3. Append the contents of `INDEX.md` to the table
4. Skill files in `skill-packs/engineering/skills/` load on-demand
