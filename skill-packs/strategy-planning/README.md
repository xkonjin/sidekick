# Strategy & Planning Skill Pack

7 skills for strategic thinking: quarterly and annual planning, scenario analysis, competitive positioning, career development, relationship mapping, and initiative tracking.

## Skills

| Skill              | What it does                                                                            |
| ------------------ | --------------------------------------------------------------------------------------- |
| quarterly-plan     | Builds a full quarterly plan from priorities, projects, capacity, and deadlines.        |
| annual-review      | Synthesizes the year's outputs into a narrative review with forward implications.       |
| scenario-planner   | Maps 2-4 alternative futures with probability, impact, and response plans.              |
| strategy-canvas    | Builds a Playing to Win canvas: Where to Play, How to Win, Capabilities Required.       |
| career-compass     | Maps current skills vs desired role, identifies gaps, suggests learning priorities.     |
| relationship-map   | Builds a visual map of allies, sponsors, dependencies, and tensions.                    |
| initiative-tracker | Tracks multi-team initiatives spanning multiple project files with status and blockers. |

## Install

Say "add the strategy and planning skills" during any conversation, or manually:

1. Open your `CLAUDE.md`
2. Find the `## Skills` table
3. Append the contents of `INDEX.md` to the table
4. Skill files in `skill-packs/strategy-planning/skills/` load on-demand
